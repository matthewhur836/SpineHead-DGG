
Plenum Installation instructions


(1) Open Mathematica

(2) Type $BaseDirectory and note the name of the directory identified.

	Typical locations are: 

	/Library/Mathematica/    on Mac OSX
	/usr/share/Mathematica/  on Unix
	\Documents_and_Settings\All_Users\Application_Data\Mathematica on Windows
	
	For more details on system-specific folders see the Mathematica Documentation
		
(3) Copy all the packages (.m files) in the distribution into the folder
	Applications within the directory identified above.
	
	If the "Applications", folder does not exist, create it first.

(4) To load Plenum:
	<<Plenum.m

(5) For testing , Run the demo file: demo.nb
